# Pipeline de Análise Financeira

Projeto desenvolvido em Python simulando um pipeline de dados financeiros, incluindo etapas de carregamento, limpeza, validação e análise.

## Funcionalidades
- Carregamento de dados  
- Limpeza de dados (remoção de inconsistências)  
- Validação de registros  
- Análise de valores financeiros  
- Geração de relatório  

## Tecnologias utilizadas
- Python  
- Pandas  

## Como executar
1. Faça upload do arquivo financeiro_pipeline.csv  
2. Execute o script pipeline_financeiro.py  
3. O sistema irá gerar um relatório automaticamente
